# roguelike

## Prerequisite

*
1. Rust
2. Cargo
3. libtcod
  3.1. brew install sdl
  3.2. git clone https://github.com/podiki/libtcod-mac.git
  3.3. cd libtcod-mac
  3.4. make -f makefiles/makefile-osx release
